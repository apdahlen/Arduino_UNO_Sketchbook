char  delim

This function serves to quickly parse a string and return the desired field.

Example:

1 sprintf(line, "%s", "$GPGGA,123519,4807.038,N,01131.000,E,1,08,0.9,545.4,M,46.9,M,,*47");
2 num_fields = line_parser(line, ',');
3 printf("\nThe string \"%s\" contains %d fields. The individual fields are: \n", line, num_fields);
4 for (i = 1; i < num_fields; i++){
5  num_char_in_field = get_field(i, line);
6  printf("\tfield %d with length %d = %s\n",i, num_char_in_field, line );
7 }

This script will output:

This function could have been designed using an array of pointers to hold the location of each field. This would have made the results easier to use. This was not done to keep the array as small as possible - an important decision for use with smaller computers such as Arduino.
