Detailed Description

Observe that the Arduino SPI library "allows you to communicate with SPI devices, with the Arduino as the master device." The code contained in this library allows you to use the Arduino as a slave or as a master.

Author
Aaron P. Dahlen
Date
18 Feb 2015

add examples showing how to use the Arduino as a slave and as a master

See Also
http://arduino.cc/en/Reference/SPI
